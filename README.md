# Car Rental Management System

A full-stack web application for managing vehicle rentals, built with Flask and SQLAlchemy.

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite with SQLAlchemy ORM
- **Frontend:** HTML5, CSS3, JavaScript

## Features
- User authentication with hashed passwords
- Vehicle inventory management
- Full booking lifecycle — Reserve → Active → Returned
- Payment tracking per booking
- Dashboard with real-time revenue analytics

## How to Run
Clone the repo and run:
pip install -r requirements.txt
python app.py

Then open http://127.0.0.1:5000
