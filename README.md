# Car Rental Management System

A full-stack web application for managing vehicle rentals, built with Flask and SQLAlchemy.

## Live Demo

🌐 https://car-rental-ashik.onrender.com

## Tech Stack

- Backend: Python, Flask
- Database: SQLite with SQLAlchemy ORM
- Frontend: HTML5, CSS3, JavaScript
- Authentication: Flask Sessions, Werkzeug Password Hashing
- Deployment: Render

## Features

- User Registration and Login
- Secure Password Hashing
- Session-Based Authentication
- Dashboard with Revenue Analytics
- Car Management (Add, Edit, Delete)
- Customer Management
- Booking Management
- Payment Tracking
- Car Return Management

## Run Locally

```bash
git clone https://github.com/mohamedashik110/Car_Rental_Project.git
cd Car_Rental_Project
pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
